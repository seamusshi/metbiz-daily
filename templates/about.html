项目简介

MetBiz Daily Report 是一个自动化气象行业情报收集系统，每日自动扫描招标信息、政策动态、竞品动态和技术趋势，生成结构化日报并发布。

监控范围

  • 招标信息 - 各省气象局招投标、中标公告
  • 政策动态 - 中国气象局及各省政策文件、规划通知
  • 竞品动态 - 航天宏图、华云、墨迹天气等企业动态
  • 技术趋势 - 气象AI、大数据、卫星遥感等新技术应用

数据来源

  • 中国政府采购网 (ccgp.gov.cn)
  • 中国气象局政府采购网
  • 各省公共资源交易中心
  • 行业新闻与公开报道

更新频率

每日上午 9:17 (UTC+8) 自动更新

技术栈

  • Python + Jinja2 模板引擎
  • GitHub Actions 定时任务
  • GitHub Pages 静态托管
